Section 4:

Streams


This last section will go over reading and writing files being passed through the server. The code that we will use will be very similar to the code used in the last section.

var fs = require('fs');

var http = require('http');

http.createServer(function(request, response){

  var newFile = fs.createWriteStream("coolFile.txt");

  request.pipe(newFile);

  request.on('end', function(){
    response.end('upload complete');
  });
}).listen(8080);

From the command line type:

> curl --upload-file coolFile.txt http://localhost:8080

The fantastic part about the above code is that you can read and write files extremely fast. As the file is being uploaded, the server is putting it together piece by piece instead of waiting for the whole file to finish. So basically, simultaneous reading and writing. That's due to pipe, which allows the file to be read "chunk" by "chunk".

And that's it for now! After going through this How-To-Guide you should be able to:

  • Use EventEmitters for custom listeners
  • Create a server and listen for requests
  • Read and Write from files / File Uploading
